# FoodWise Smart Meal Planner

FoodWise is an interactive meal planner that helps users generate meal ideas, create weekly meal plans, build grocery lists, track meal categories, estimate food cost, and reduce food waste.

This project is built using HTML, CSS, JavaScript, and LocalStorage. It can be deployed directly using GitHub Pages without GitHub Actions.

## Live Demo

Live project: https://danishzulkafli.github.io/foodwise-smart-meal-planner/

## Features

- Smart meal idea generator
- Meal goal selection
- Meal time selection
- Custom meal creation
- Weekly meal plan
- Grocery list generator
- Grocery export as CSV
- Meal category breakdown
- Estimated food cost
- Healthy meal counter
- Search meals
- Filter by meal type
- Delete meal
- Reset data
- LocalStorage persistence
- Responsive interface
- GitHub Pages deployment without GitHub Actions
