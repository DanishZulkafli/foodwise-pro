# FoodWise Pro — Smart Meal Planner

FoodWise Pro is an interactive meal planner that helps users generate meal ideas, create weekly meal plans, build grocery lists, track nutrition goals, estimate weekly food cost, and reduce food waste.

This project is built using HTML, CSS, JavaScript, and LocalStorage. It can be deployed directly using GitHub Pages without GitHub Actions.

Live project: https://danishzulkafli.github.io/foodwise-pro/

## Purpose

This project was created as a practical and creative portfolio project that helps users plan meals, reduce food waste, manage grocery needs, and build healthier food habits.

It demonstrates frontend development, form handling, LocalStorage, filtering, search, dashboard statistics, data export, and user-centered design.
